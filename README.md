# NIKE ZONE ⚡

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge)](https://nike-shoe-zone.vercel.app/)

A premium, minimalist e-commerce storefront reimagined for modern performance and lifestyle. Built with a high-contrast design system, smooth micro-animations, and fluid responsive layouts.

Developed & Maintained by **Hemant** ([@Hemant5241](https://github.com/Hemant5241))

---

## 🎨 Design & Aesthetic Philosophy
Nike Zone is engineered with a luxury minimalist aesthetic:
* **High Contrast Colorway** — Pure whites, deep blacks, and subtle neutral grays accented with premium micro-interactions.
* **Typographic Authority** — Bold uppercase layouts, heavy tracking, and clean monospaced alignments.
* **Fluid Micro-Animations** — Powered by Framer Motion and custom mouse-following physics to feel responsive and alive.
* **Functional Modern UI** — Rounded floating-label inputs, sticky backdrop-blurred headers, and custom-scrolling cart overlays.

---

## ⚙️ Tech Stack
* **Core Framework** — [React](https://react.dev/) (v18)
* **Build Tooling** — [Vite](https://vite.dev/) (v6)
* **Styling Engine** — [Tailwind CSS](https://tailwindcss.com/) (v3)
* **Vector Icons** — [Lucide React](https://lucide.dev/)
* **Motion Graphics** — [Framer Motion](https://www.framer.com/motion/) & [React Mouse Follower](https://github.com/h-richard/react-mouse-follower)
* **Routing** — [React Router DOM](https://reactrouter.com/) (v7)
* **Deployment** — [Vercel](https://vercel.com/) (with custom SPA catch-all rewrites)

---

## 🚀 Key Features
1. **Interactive Navigation** — Sticky glassmorphic header with real-time cart badge system.
2. **Premium Product View** — Interactive cards with hover scale, rotation, quick-view controls, and enlarged add-to-bag triggers.
3. **Immersive UI Polish** — High-end input masks, floating label alignment, and customized minimalist scrollbar architectures.
4. **Fluid Responsiveness** — Seamlessly adapts from massive desktop monitors down to mobile viewports.

## 💻 Local Setup & Installation

### Prerequisites
* **Node.js** (v18 or higher recommended)
* **npm** (v9 or higher)

### 1. Clone the Repository
```bash
git clone https://github.com/Hemant5241/nike-zone.git
cd nike-zone
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start Development Server
```bash
npm run dev
```

### 4. Build for Production
```bash
npm run build
```
